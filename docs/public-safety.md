# 공개 안전성 검토

## Drive에서 확인한 비공개 후보

- `build/`: 로컬 빌드 산출물
- `.dart_tool/`, `.idea/`: 로컬 개발 환경 파일
- `firebase.json`: 배포/프로젝트 환경 설정 가능성
- `.flutter-plugins*`: 로컬 플러그인 경로 정보 가능성
- `리뷰_크롤링.ipynb`: 크롤링 출처와 재배포 조건 확인 필요

## 이번 조치

Drive 자료는 참고만 하고 GitHub에 추가하지 않았습니다. README에는 현재 구현 상태와 공개 제외 기준을 명시했습니다.

## 향후 공개 기준

- 공공데이터는 출처, 라이선스, 갱신일을 명시한 뒤 사용합니다.
- 크롤링 데이터는 서비스 약관과 개인정보 포함 여부를 검토한 뒤 샘플화합니다.
- Firebase, API key, 배포 설정 파일은 환경 변수 또는 비공개 설정으로 분리합니다.
