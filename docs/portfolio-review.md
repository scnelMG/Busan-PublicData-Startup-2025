# 포트폴리오 리뷰

## 현재 평가

이 저장소는 공공데이터 활용 앱 아이디어의 초기 Flutter 코드베이스입니다. 현재 코드만 보면 Flutter 기본 템플릿에 가까우므로, 완성형 프로젝트로 주장하기보다는 공공데이터 서비스 프로토타입의 출발점으로 설명하는 것이 안전합니다.

## 강점

- Flutter 기반 멀티플랫폼 프로젝트 구조가 있습니다.
- 공공데이터 창업경진대회 맥락이 명확합니다.
- Drive에 리뷰 크롤링과 Firebase 설정 흔적이 있어 서비스화 시도 근거가 있습니다.

## 보완 필요

- 실제 서비스 화면 구현
- 공공데이터 출처와 문제 정의 문서화
- 인증/라우팅 파일 구현
- 크롤링 데이터 공개 기준 정리
- 앱 화면 캡처 또는 시연 GIF 추가

## 제출 전략

은행 IT, 금융권, 공기업 전산직 포트폴리오에서는 이 프로젝트를 메인으로 내세우기보다 `공공데이터 기반 앱 프로토타입` 항목으로 짧게 소개하는 것이 좋습니다. 메인 프로젝트는 금융 데이터 분석, 위험 관리, 추천/매칭 프로젝트를 우선 배치하는 편이 더 강합니다.
